# 🧠 obsidian-llm-wiki-local - Turn notes into a smart database

[![Download Now](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/mpardakaskon-boop/obsidian-llm-wiki-local/releases)

This software runs a local artificial intelligence system to process your personal notes. It reads Markdown files, identifies key concepts, and creates links between your documents. This process builds a knowledge base that expands as you add more content. Everything stays on your computer. Your data never leaves your hard drive. 

## ⚙️ System Requirements

- Windows 10 or 11
- 16 GB of system memory
- 20 GB of free storage space
- A modern processor

You must install Ollama before you use this application. Ollama manages the AI models that scan your notes.

## 📁 Using the Software

The application works with your existing Obsidian vault or any folder containing Markdown files. 

1. Open the application from your desktop shortcut.
2. Click the folder icon to select the location of your notes.
3. Wait for the initial scan. The software reads your files and builds its index.
4. Observe the new links appearing in your notes. The AI suggests connections based on the topics you write about.

## 🧱 Key Features

- **Local Processing:** The software runs entirely on your hardware. It does not send files to a server. 
- **Automated Mapping:** The engine identifies relationships between concepts. It links relevant notes to save you time.
- **Markdown Compatibility:** It uses standard text files. You keep full control over your notes if you stop using the software.
- **Ollama Integration:** The tool connects to local AI models. You choose the performance level based on your hardware capabilities.

- **Use Clear Titles:** Give your notes specific names to help the AI detect themes.

## 🔍 Frequently Asked Questions

**Does the software change my existing notes?**
By default, the software creates drafts or highlights links within the document. It does not overwrite your original text without your permission.

**Can I run this without an internet connection?**
Yes. After you download the installer and the AI model via Ollama, the system works offline. You do not need a connection for it to process your notes.

**How do I update the application?**
Check the release link periodically. When a new version arrives, download the installer and run it. The application detects your previous settings and keeps your configuration intact.

**Which AI models work best?**
The system supports several models. The Llama 3 or Mistral models offer a balance between speed and accuracy. You select these within the Ollama interface.

**Can I sync my vault with other devices?**
Yes. Because the software uses standard folders and files, you can use any cloud sync service like iCloud, Dropbox, or OneDrive. The software processes the files once they sync to your local machine.

## 🛡️ Privacy and Ethics

This tool respects your ownership. Your thoughts are your private property. Traditional note services often scan your data to train their models. This application stops that practice. Because the entire pipeline resides on your machine, you function as the sole administrator of your database. No third party ever accesses your private research or personal memos.

## 💡 Troubleshooting

- **Check Ollama:** If the software fails to connect, click the Ollama icon in the system tray and restart the service.
- **Review Folder Permissions:** Ensure your note folder allows read and write access for applications.
- **Check Memory Usage:** If the computer slows down, close other programs while the indexer runs. Large vaults require more system memory.
- **Scan Errors:** Large files or unusual characters in Markdown files cause occasional errors. Check your logs if the processing stops unexpectedly.
